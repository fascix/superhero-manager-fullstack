NEXT STEPS TO RUN:

1. Install MongoDB:
   - Local: Download from mongodb.com
   - Cloud: Create free cluster on MongoDB Atlas

2. Backend Setup:
   cd backend
   cp .env.example .env    # Edit if using Atlas
   npm install
   npm run build
   node dist/utils/seedDatabase.js   # Import heroes
   npm run dev             # Start on port 5000

3. Frontend Setup (new terminal):
   cd frontend
   npm install
   npm run dev             # Start on port 5173

4. Create Admin User (curl or Postman):
   POST http://localhost:5000/api/auth/register
   Body: {"username": "admin", "password": "admin123", "role": "admin"}

5. Access Application:
   Open http://localhost:5173
   Login with created credentials
   Explore the hero management system!
